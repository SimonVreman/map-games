A game that challenges you to identify the countries of the most common European bollards on a map. Similar bollards are grouped together, pay attention to the expected number of countries and start with the most likely candidate.

Resources from [GeoHints](https://geohints.com/meta/bollards) and [Plonk It](https://www.plonkit.net/) have been used.
