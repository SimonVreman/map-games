A simple streak-based game to memorize which Brazilian telephone code corresponds to which locations. Ideal for those who want to gain an edge in Brazil region guessing.

The codes are organized by region, you get feedback when you guess incorrectly. Read more about these dialing codes on [Wikipedia](https://en.wikipedia.org/wiki/List_of_dialling_codes_in_Brazil).
