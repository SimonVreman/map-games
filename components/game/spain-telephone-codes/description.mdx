# Spain Telephone Codes

A simple streak-based game to memorize which Spainish telephone code corresponds to which locations. Ideal for those who want to gain an edge in Spain region guessing.

The codes are organized by region, you get feedback when you guess incorrectly.Read more about these dialing codes on [Wikipedia](https://en.wikipedia.org/wiki/Telephone_numbers_in_Spain).
